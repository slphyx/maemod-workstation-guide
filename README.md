# MAEMOD Research Workstation User Guide

This repository hosts the published website for the **MAEMOD Research Workstation User Guide**.

The guide provides documentation for researchers, students, and research software engineers using the MAEMOD shared Linux workstation.

Topics include:

- Connecting to the workstation
- Basic Linux commands
- File storage and project organisation
- R, Python, and CmdStan workflows
- R package management using Conda and `renv`
- Slurm job submission
- Troubleshooting
- Workstation administration
